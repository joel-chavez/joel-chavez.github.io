Nbrvalues (2023) | Repository for Statistical package for R program. This library returns measures of central tendencies for neighboring polygons in spatial features -- e.g., the median oil and gas production of a county's neighboring counties. To install this package, use the following chunk of code: 'devtools::install_github("nbrvalues/nbrvalues")'
